Diseño y desarrollo de páginas web

Con mi servicio de diseño y desarrollo de páginas web, tendrás acceso a una plataforma personalizada diseñada en Figma y desarrollada en Wordpress. Obtendrás una página funcional, estética y que genere ventas. Además, te enseñaré cómo usar la plataforma para que puedas administrar tu sitio web por tu cuenta. Para asegurarnos de que tu sitio web sea un éxito, instalaré herramientas de analítica web como Google Analytics y Hotjar. De esta manera, podrás monitorear y analizar el tráfico de tu sitio web y tomar decisiones informadas.

1.Qué incluye el servicio?

-Archivo figma no editable del proyecto

-Sitio wordpress administrado por el cliente

-1 página web ( página de inicio, página de conócenos, página de blog, página de servicios, landing page ( según se requiera)

-Asesoría para utilizar la plataforma de wordpress y buenas prácticas

-Asesoría para utilizar herramientas de analítica web: Google analytics y Hotjar

-Asesoría para lograr un buen SEO y posicionamiento en Google

-Mantenimiento de la página por 6 meses ( NOTA: mantenimiento incluye ajustes generales y configuraciones del sitio web mas no CAMBIOS en el diseño de la página)

2.Qué NO incluye el servicio?

-Dominio y hosting del sitio. Esto lo adquiere el cliente. Sin embargo, se puede asesorar al cliente en este camino para escoger la mejor solución para la página web

4.Cuánto es el costo del servicio?

El costo es de $300-500 dependiendo de las necesidades del cliente ( el costo varía por el número de páginas adicionales y por el diseño)
